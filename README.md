stunning-zintle
Stunning Zintle is a creative repository built to explore innovative web design and coding techniques. It features experiments with responsive layouts, interactive components, and clean project organization. The goal is to provide a foundation for learning, sharing, and expanding modern development skills. Build a modern, responsive web application called AI Workplace Productivity Assistant.

Purpose: Help IT support technicians automate daily tasks.

Features to include:

Smart Email Generator: Input rough notes, select tone [formal, friendly, persuasive], output editable email draft.
Meeting Notes Summarizer: Paste notes, output summary, decisions, action items with owners and deadlines.
AI Task Planner: Input tasks and deadlines, output prioritized daily/weekly schedule.
Requirements:

Modern SaaS-style dashboard with sidebar navigation
Responsive design for mobile and desktop
Each feature has input box, output box, and "Copy" button
Editable AI outputs
Responsible AI disclaimer at the bottom: "AI can make mistakes. Verify outputs before use."
Clean, professional UI with TailwindCSS
Tech: Use React and Tailwind. Mock the AI calls for now with a placeholder function
